# Earthquakes

### Valid queries for Earthquake API:
`date, time, latitude, longitude, depth, magnitude_md, magnitude_ml, magnitude_mw, location, state, google_maps`
```sh
$ curl https://www.ayberkenis.online/v1/earthquakes
```
### Latest earthquake:
```sh
$ curl https://www.ayberkenis.online/v1/earthquakes/latest
```
### All earthquakes (500):
Latest earthquake:
```sh
$ curl https://www.ayberkenis.online/v1/earthquakes/all
```

# Exchange
### Valid queries for Exchange API
`currency_code, currency_name, currency_buy, currency_sell, currency_high, currency_low, currency_change, time`
```sh
$ curl https://www.ayberkenis.online/v1/exchange
```
### All exchanges
```sh
$ curl https://www.ayberkenis.online/v1/exchanges/
```
